Project 2: Event-Driven E-Store Simulation

Summary: For this project I created a small desktop program in Java that simulates an online store. The app lets a user add items to a cart, update quantities, and go through a basic checkout. It also calculates tax and gives a receipt at the end. This was mainly practice in event-driven programming and using Swing to build a working GUI.

What I Did

  • Built a Java Swing interface with windows for browsing items and managing the shopping cart.
  • Wired up event handlers so that buttons (like “Add to Cart” or “Checkout”) actually triggered the right actions.
  • Made the program output a receipt that included item totals, sales tax, and a final balance.

Tech & Skills

  • Java with Swing (GUI design and event handling)
  • Event-driven programming basics
  • Working with transactions and simple logging
